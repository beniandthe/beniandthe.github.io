# Homebrew Helper
www.homebrew-helper.com

RPG Toolkit Pro is a lightweight toolkit for game masters, tabletop designers, and RPG creators who want fast, practical tools for planning and organizing campaigns.

It includes standalone tools for:

- XP and progression planning
- encounter building
- loot and reward design
- quest generation

Pro users also get access to **Campaign Hub**, a premium workspace that links those tools together into a single campaign structure.

---

## Features

### Free tools
All users can use and save standalone projects for:

- **Progression Planner**  
  Build XP curves, compare leveling styles, and estimate campaign pacing.

- **Encounter Builder**  
  Create and balance encounters using party level, role mix, terrain, enemy type, and wave structure.

- **Reward Designer**  
  Generate more useful loot by combining rarity, source, theme, bundle style, and reward guidance.

- **Quest Builder**  
  Create practical quests with hooks, twists, consequences, alternate resolutions, and faction pressure.

### Pro features
Pro unlocks **Campaign Hub**, which allows users to:

- create campaign workspaces
- link quests, encounters, loot, and progression plans into one campaign
- organize prep in a cleaner, more structured way
- manage campaign-linked content separately from standalone saved projects

## How it works

Users can start with standalone tools for free.  
When they upgrade to Pro, they unlock Campaign Hub, which acts as an organizational layer across the toolkit.

This keeps the free experience simple and useful, while giving power users a more complete campaign workflow.
